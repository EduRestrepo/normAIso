# Roadmap Tecnico

## Fase 1. Base operativa

Estado:

- completada en gran parte

Incluye:

- PostgreSQL
- Docker
- marcos y controles configurables
- evaluaciones
- evidencias manuales y referenciadas
- resultados automaticos base
- hallazgos
- planes de accion
- bitacora

## Fase 2. Integraciones reales

Objetivos:

- integracion real con proveedor IA
- SharePoint real
- OneDrive real
- extraccion de texto PDF/DOCX/XLSX
- OCR para escaneados
- lectura de logs y CSV

## Fase 3. Gobierno y seguridad

Objetivos:

- autenticacion real
- RBAC por rol y por evaluacion
- secretos cifrados
- bitacora ampliada
- aprobaciones y workflow

## Fase 4. Escalabilidad

Objetivos:

- colas para procesamiento IA
- workers asincronos
- almacenamiento de objetos
- observabilidad
- tareas programadas de reevaluacion

## Fase 5. Inteligencia avanzada

Objetivos:

- RAG sobre normativa y evidencias
- comparativas historicas
- matrices entre marcos
- recomendacion automatica de nuevas evidencias
- scoring de calidad documental
